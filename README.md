## 📖 项目简介 (Introduction)

本项目是一个基于 **Ren'Py** 引擎开发的视觉小说（Galgame）。

故事发生在初秋的长春，主角 **木子米** 怀揣着对大学生活的憧憬步入校园。在这里，他发生了各种各样有趣的内容

从军训的烈日暴晒，再到各种奇妙的邂逅，一段充满欢笑与悸动的大学生活就此展开。

## 🎭 主要角色 (Characters)

* **木子米 (Protagonist)**
    * 土木工程系新生，性格随和（有点木头），吐槽役担当。擅长在混乱的局势中生存（比如食堂抢饭）。
* **素织 (Suzhi)**
    * 外表清冷倔强，扎着马尾辫。军训时的标兵，私下里却是个会因为草莓牛奶撒了而暴走的傲娇少女。
* **凌宁 (Lingning)**
    * 自称“寻觅真理的贵族学者”，木子米的室友。无论是军训罚站还是食堂抢饭，都要保持绝对的“优雅”。
* **白墨萱 (Baimoxuan)**
    * 被称为“结构力学的魔女”，经常在图书馆进行奇怪的“炼成阵”研究（其实是堆书）。

## 🛠️ 如何运行 (How to Run)

你可以从 Actions 板块直接下载构建完成的PC版本

如果你想在本地运行或测试本项目：

1. **安装环境**：下载并安装最新版的 [Ren'Py SDK](https://www.renpy.org/latest.html)。

2. **克隆项目**：
   ```bash
   git clone [https://github.com/aquasofts/Snow-Love.git](https://github.com/aquasofts/Snow-Love.git)
   ```

3. **导入项目**：
   * 打开 Ren'Py 启动器。
   * 点击 `Preferences` -> `Projects Directory`，选择你克隆下来的文件夹路径。
   * 回到主界面，点击 `Snow Love`。

4. **启动游戏**：点击 `Launch Project`。
